## Using Built-In Modules:

To see the complete list of built-in modules visit https://nodejs.org/api/

#### FS and OS used in below example
app.js
```
const fs= require('fs');
const os= require('os');

var user = os.userInfo();

fs.appendFileSync('greetings.txt','Hello '+user.username +' !');

fs.appendFileSync('greetings.txt',`Hello ${user.username} !');

```
